# 6. 기술 스택

## 6.1 Frotend
- Language: JavaScript,HTML,CSS
- Framework: React 19
- Build Tool: Vite

<br/>

## 6.2 Backend
- Langauage: Java 17
- Framework: SpringBoot 4.0.0, Spring Security
- Build Tool: Gradle
- Database: MySql
- Cache: Redis
- ORM: Spring Data JPA, QueryDsl 5.0.0
- Migration Tool: flyway
- Infra: AWS S3

<br/>
